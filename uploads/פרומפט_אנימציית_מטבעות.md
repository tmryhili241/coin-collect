# פרומפט לקלוד — רכיב אנימציית מטבעות ל-HIT

## הקשר
אני בונה קורס דיגיטלי בשם "עולים שלב" בפלטפורמת **HIT**.
מנגנון הגמול בקורס מבוסס על **מטבעות זהב פיראטיים**.

**מגבלת פלטפורמה:** ב-HIT כל מסך עומד בפני עצמו — אין header קבוע על כל המסכים.
HTML מוכנס דרך רכיב **"פסקה מעוצבת"** — תומך ב-HTML/CSS/JS מלא.

**הפתרון האדריכלי:** localStorage כגשר בין שני רכיבים עצמאיים:
- **רכיב Counter** — מוכנס ידנית בכל מסך, קורא מה-localStorage ומציג סכום עדכני
- **רכיב אנימציה** — מוכנס במודל, מפעיל טיסת מטבעות וכותב ל-localStorage

---

## רכיב 1 — Counter (מוכנס בכל מסך)

### תצוגה
- מסגרת קומפקטית עם border זהוב-חום (סגנון פיראטי, לא תיבת אוצר)
- פורמט: **[תמונת מטבע] × [מספר]**
- גודל מוצע: ~180×60px
- בטעינת הדף: קורא את הסכום מ-localStorage ומציג אותו מיד
- אם localStorage ריק: מציג 0

### אירועים
- מאזין לאירוע `storage` — כשרכיב האנימציה מעדכן את localStorage, ה-counter מתרענן אוטומטית עם אפקט count-up קצר

---

## רכיב 2 — אנימציה (מוכנס במודל)

### אנימציית צבירה
- בטעינת המודל: מטבעות עפים מקצוות המסך ומתאספים לתוך המסגרת
- מספר המטבעות הטסים מוגדר דרך **data attribute** על הרכיב: `data-coins="3"`
- המטבעות טסים בנתיב קשתי/אורגני (לא קו ישר), כל מטבע בנתיב מעט שונה
- המטבעות מרחפים **מעל כל שאר התוכן** (z-index: 9999, position: fixed)
- בסיום האנימציה: מוסיף את המטבעות ל-localStorage, ה-counter מתרענן

### עיצוב
- רקע שקוף לחלוטין
- פונט: Rubik (Google Fonts), bold, RTL
- צבע טקסט ומסגרת: גוונים זהוב-חום (#8B6914, #3D2B1F)
- אנימציה: חלקה, ~1.5 שניות, מטבעות מסתובבים קלות תוך כדי טיסה

---

## דרישות טכניות

- **שני קבצי HTML נפרדים**, כל אחד self-contained (HTML + CSS + JS inline)
- ללא תלויות חיצוניות (חוץ מ-Google Fonts)
- תמונת המטבע: base64 מוכנסת ישירות לתוך הקוד (אצרף PNG)
- fallback: אם localStorage חסום → sessionStorage → variable בזיכרון
- הקוד צריך לעבוד בתוך `<div>` בתוך דף HIT, לא כדף עצמאי
